# Документация проекта

## Структура документации

### Актуальная документация (в корне проекта)
- **README.md** - Основной файл с описанием проекта
- **PROJECT_DOCUMENTATION.md** - Полная документация проекта
- **QUICK_DEPLOY.md** - Быстрая инструкция по деплою
- **DEPLOYMENT_STEPS.md** - Подробная пошаговая инструкция по деплою
- **LOGS_GUIDE.md** - Руководство по работе с логами
- **COMMISSION_DETECTION_LOGIC.md** - Логика обнаружения комиссии
- **SERVER_OPTIMIZATION_AND_SAFETY.md** - Оптимизация и безопасность

### Архивная документация
Папка `archive/` содержит устаревшие файлы документации, сохраненные для справки:
- Старые руководства по настройке
- Исторические заметки о разработке
- Устаревшие инструкции по оптимизации

## Бекапы

Все бекапы версий проекта хранятся в папке `backup/` в корне проекта. Каждый бекап содержит:
- `BACKUP_INFO.txt` - Информация о бекапе
- Все исходные файлы проекта на момент создания бекапа

**ВАЖНО:** Старые бекапы не удаляются для возможности отката к предыдущим версиям.
