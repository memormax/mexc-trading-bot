# Тестирование локального сервера

## Запуск сервера

```bash
cd unified-bot
npm run build
npm start
```

## Проверка работы

1. **Приветственная страница:** http://localhost:3002/
   - Должна открыться страница с приветствием и кнопкой "Открыть торгового бота"

2. **Торговый бот:** 
   - http://localhost:3002/flip - открывает торгового бота
   - http://localhost:3002/flip/ - также открывает торгового бота

## Структура маршрутов

- `/` → Приветственная страница (welcome.html)
- `/flip` → Редирект на `/flip/`
- `/flip/` → Торговый бот (index.html)
- `/api/*` → API endpoints
- `/reports.html` → Страница отчетов
- Все остальные пути → Торговый бот (index.html)
