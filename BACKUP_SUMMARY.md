## Восстановление из бекапа

```bash
# 1. Скопировать файлы из бекапа
cp -r backup/backup_2026-02-05_16-01-32/unified-bot/* ./

# 2. Установить зависимости
npm install

# 3. Собрать проект
npm run build

# 4. Запустить
npm start
```

- Бекап не включает данные пользователей (`data/`) для безопасности
- Для полного восстановления нужно также скопировать `data/` (если есть)
