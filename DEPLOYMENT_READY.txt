================================================================================
                          БЫСТРЫЙ СТАРТ НА СЕРВЕРЕ
================================================================================

1. Загрузить проект на сервер
2. cd unified-bot
3. npm install
4. npm run build
5. mkdir -p data logs
6. pm2 start ecosystem.config.js
7. pm2 save
8. pm2 startup

Подробности: DEPLOYMENT_GUIDE.md
